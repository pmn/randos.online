randos.online

A collection of randos from the internet

To add the counter to your GitHub profile, use this markdown (be sure to replace YourUserName with your GitHub username):
[![some alt text](https://www.randos.online/u/YourUserName)](https://randos.online/u/YourUserName/next)

Configuration options:

  • Theme: choose a color theme by including ?theme=color after your username. Current color options are blue, red, green, and hotdog.
